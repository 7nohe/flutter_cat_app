# flutter_cat_app

- Flutter 2.10.4

## Setup

### Get Unsplash Client ID

[Join Unsplash Developers](https://unsplash.com/oauth/applications)

### Create env file

```
$ cp .env.example .env
```

Edit .env

```
UNSPLASH_CLIENT_ID=<YOUR_CLIENT_ID>
```


### Install packages

```
$ flutter pub get
```
